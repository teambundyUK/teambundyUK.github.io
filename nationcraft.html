What Is Nationcraft?

The Nationcraft SMP is a whitelisted vanilla survival server for Minecraft 1.17.1. It also has quality of life features like multiplayer sleep and a dynamic web map to improve your experience!

How To Join:

Join discord

If you would like to join the nationcraft SMP, join our discord server and agree to the rules. You can then go to the SMP applicants category and apply.
